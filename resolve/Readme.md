DNS Resolver
============
These scripts resolves names to addresses and serves a few purposes:
1.  Detect if a destination is censored. Most censors start blocking websites by hijacking DNS.
2.  Force enable IPv6 for websites that do not publish AAAA records.
3.  Get around IP address based blocks.

Bypassing IP blocks
-------------------
Yes, it's possible! Censors are often highly targeted with IP blocks, usually blocking individual addresses, but the IPv6 address space is massive. To bypass it just use a different address.
